## Features

- Limits editor to one block of text. Suitable for brief content or titles.

</PackageInfo>

## Installation

```bash
npm install @sewell_stephens/late-break
```

## Usage

```tsx
import { SingleLinePlugin } from '@sewell_stephens/late-break/react';

const plugins = [
  // ...otherPlugins
  SingleLinePlugin,
];
```
